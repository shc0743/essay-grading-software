<h1>隐私政策</h1>

<p><strong>生效日期：</strong>2025年10月17日</p>

<h2>概述</h2>
<p>Essay Grading Software（https://essay-grading-software.clspd.top/）是一个基于AI服务的作文自动批改工具。我们郑重承诺保护您的隐私，并采用以下数据处理方式。</p>

<h2>核心隐私原则</h2>
<ul>
    <li><strong>本地处理：</strong>所有API密钥和作文内容仅在您的本地浏览器中存储和处理</li>
    <li><strong>无服务器存储：</strong>我们采用serverless架构，不在任何服务器上存储您的数据</li>
    <li><strong>最小化收集：</strong>除必要的流量统计外，不收集任何个人身份信息</li>
</ul>

<h2>我们收集的信息</h2>
<p>我们仅收集最少量的必要信息：</p>
<ul>
    <li><strong>技术信息：</strong>通过Cloudflare Web Analytics收集匿名的网站使用统计信息（如访问量、页面浏览量等）</li>
    <li><strong>本地存储：</strong>您的浏览器本地存储中保存的API密钥和作文内容（我们无法访问）</li>
</ul>

<h2>我们不收集的信息</h2>
<p>我们明确声明不收集以下信息：</p>
<ul>
    <li>个人身份信息（姓名、邮箱、电话号码等）</li>
    <li>您输入的作文内容（仅在您的浏览器本地处理）</li>
    <li>您的API密钥（仅在您的浏览器本地存储和使用）</li>
    <li>任何能够识别您个人身份的信息</li>
</ul>

<h2>数据存储和处理</h2>
<ul>
    <li><strong>本地存储：</strong>所有API密钥和作文内容仅存储在您的浏览器本地</li>
    <li><strong>AI服务调用：</strong>作文内容通过您提供的API密钥直接发送到您选择的AI服务提供商</li>
    <li><strong>无后端传输：</strong>我们不通过自己的服务器中转任何用户数据</li>
</ul>

<h2>第三方服务</h2>
<ul>
    <li><strong>Cloudflare Analytics：</strong>用于网站流量统计，仅收集匿名使用数据</li>
    <li><strong>AI服务提供商：</strong>您自行选择的OpenAI兼容API服务商，其数据处理受其隐私政策约束</li>
</ul>

<h2>数据安全</h2>
<ul>
    <li>所有AI API调用均通过HTTPS加密传输</li>
    <li>您的API密钥仅存储在浏览器本地，不会发送到我们的服务器</li>
    <li>我们采用现代Web安全标准保护网站访问安全</li>
</ul>

<h2>您的权利</h2>
<ul>
    <li>随时清除浏览器本地存储的数据</li>
    <li>选择不使用本服务</li>
    <li>通过浏览器设置控制本地存储</li>
</ul>

<h2>政策更新</h2>
<p>我们可能会不时更新本隐私政策。更新后的政策将在本页面发布，并更新生效日期。</p>
